#7. Financial Impact Utility

Client: Kaiser, Permanente

Role: Technical Lead

Location: Pleasanton, CA


Project Description:

The Financial Impact Utility tool allows users to provide information about patient diagnosis codes and then calculate and provides the impact on risk scores based on information provided by the user. This Tool will have to support multiple roles and should have access across regions in Kaiser. Admin User, National User and Regional User will log into the application with database credentials. The User of FIU Tool should be able to upload/submit an excel/csv file via the FIU Tool, which in turn will be used to generate an Output File.


Responsibilities:

  • Study and implement the FIU requirements provided by Kaiser
  • Understand and document the interface software requirements between the application, database and other interface applications
  • Design and Develop the FIU Tool
  • Recommend the detailed infrastructure with specifications required for rolling out the solution in development, testing, staging, production, high availability and other environments which includes hardware, Operating System, database, middleware, version management tools, software licenses etc
  • Plan, assist, guide and formulate strategy for the Development, Testing and resolution of identified issues in the FIU Tool.
  • Prepare various guidelines/documents and procedures required by Kaiser for FIU Tool
  • Developed different modules of front-end application on Web platform using Microsoft technologies; .NET Framework 4.5, Visual Studio 2012, C#, ASP.NET MVC 4.0
  • SQL Server 2012 as the backend for data storage
  • Used JavaScript, jQuery, Angular.JS, CSS, HTML for re-usable of code and to make User Interface more user friendly
  • Developed unit test cases using Microsoft Visual Studio Unit Testing framework

    • Environment:

    .NET Framework 4.5, ASP.NET MVC 4.0, ASP.Net, JavaScript, jQuery, Angular.JS, CSS, HTML, SQL Server, SOAP UI, and Web services (SOAP).
