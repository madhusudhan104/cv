Project Description:

EPUB is a format for electronic publications with reflowable text in marked up document structure with associated images for illustrations, all in a container format. Reflowable text allows the text display to be optimized for the display device used by the reader of the EPUBformatted book. OPS, Open Publication Structure specifies options for representation of the content of an electronic publication. Content components can be XML-based documents for the flowable text, together with raster or vector images representing illustrations.
